# BudgetMap

Government budget visualizer with interactive charts and AI-powered spending analysis.

## About
BudgetMap makes government budgets accessible and understandable through interactive visualizations. It transforms complex budget data into clear charts and graphs, with AI-powered analysis to highlight trends, compare allocations, and help citizens and policymakers understand how public funds are spent.

## Tech Stack
- Python
- Data visualization (Plotly, Matplotlib)
- Data processing (pandas)
- Click/Typer (CLI)

## Status
Active development - core architecture and project structure established.
